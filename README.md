Ansible Role: Visual Studio Code
================================

Role to install the [Visual Studio Code](https://code.visualstudio.com) IDE / text editor.

Requirements
------------

* Ubuntu

Example Playbook
----------------

```yaml
- hosts: servers
  roles:
     - { role: gantsign.visual-studio-code }
```
